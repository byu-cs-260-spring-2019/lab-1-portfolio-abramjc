Security Check Tool

This one I think is super cool. I thought about working in cyber security for a while (hence the security+ certification), but I decided I liked other stuff more, however I still think security stuff is pretty cool. Basically, I wrote a command line front end to the Have I Been Pwned? website, to make it more convenient to use quickly and when scripting. It can check for breaches associated with an email address, as well as securely check if a given password has been cracked. It's written to be cross platform, so it should work anywhere python 3 is available (untested). This project lives on Github. Feel free to check it out for more information, or to use it yourself.
